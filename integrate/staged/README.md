# staged/ — Draft Skills Awaiting Promotion

Draft skill concepts extracted from `integrate/raw/` sources or crystallized from sessions.
Each concept folder contains a draft `SKILL.md` and at least one `EVIDENCE.md` entry.

## Promotion Gate

A staged concept graduates to a live skill when it passes:

- **One Tier-1/2 evidence item AND one local validation** (session where the pattern worked), OR
- **Two independent Tier-1–3 evidence items** from distinct source types

See `skill-wiki/SKILL.md` for the full tier table and lifecycle rules.

## Naming Convention

```
staged/
├── <concept-name>/              ← new skill candidates
│   ├── SKILL.md                 ← draft behavioral contract
│   └── EVIDENCE.md              ← grounding citations (≥1 required for promotion)
└── <skill-name>-YYYYMMDD.md    ← session crystallizations updating an existing live skill
```
