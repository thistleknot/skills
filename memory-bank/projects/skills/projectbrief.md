# Project Brief

## Purpose
This is the skills library repository containing reusable Copilot skills for various development tasks.
Skills are modular, reusable components that encapsulate specific development patterns and workflows.

## Core Requirements
- Skills must be self-contained and reusable across projects
- Each skill must have a SKILL.md file documenting its purpose and usage
- Skills should follow consistent naming and structure conventions
- Skills should be discoverable via ~/.copilot/skills/ directory

## Out of Scope
- Project-specific implementation details
- Business logic for individual applications
- Non-reusable code patterns
