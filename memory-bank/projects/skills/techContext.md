# Tech Context

## Technologies Used
- Python 3.10 (conda env py310) on Windows
- GitHub Copilot CLI for agentic coding sessions
- ~/.copilot/skills/ directory structure for skill storage

## Development Setup
- Windows primary workstation, py310 conda env
- Skills stored in ~/.copilot/skills/ directory
- Each skill has a SKILL.md documentation file

## Technical Constraints
- Skills must be self-contained and reusable
- Documentation must be complete and clear
- Skills should follow consistent naming conventions

## Dependencies
- GitHub Copilot CLI for skill invocation
- Standard Python libraries for skill implementations
- No external dependencies required for skill discovery
