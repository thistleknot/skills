# Product Context

## Why This Exists
The skills library provides a centralized repository of reusable Copilot skills that can be applied across multiple projects. This enables consistent development patterns and accelerates development by providing pre-built solutions for common tasks.

## How It Should Work
- Skills are stored in ~/.copilot/skills/ directory
- Each skill has a SKILL.md file that documents its purpose, usage, and implementation
- Skills can be invoked by name when needed
- Skills follow consistent patterns for discovery and execution

## User Experience Goals
- Easy discovery of available skills
- Clear documentation of skill capabilities
- Consistent invocation patterns
- Reusable across different projects and contexts
